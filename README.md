# Belajar Membuat Aplikasi Android untuk Pemula  

Repository ini berisi materi dari kursus **Belajar Membuat Aplikasi Android untuk Pemula** yang diselenggarakan oleh Dicoding Indonesia. Kursus ini dirancang untuk membantu pemula memahami dasar-dasar pengembangan aplikasi Android menggunakan Android Studio.  

## Materi yang Dipelajari  

1. **Pengenalan Android Studio**: Penjelasan tentang cara menginstal Android Studio dan pengenalan komponen-komponen yang sering digunakan. Ini akan memudahkan Anda untuk mengikuti latihan selanjutnya. _(7 jam 20 menit)_  

2. **Activity**: Membuat aplikasi pertama Anda, mulai dari desain hingga logika dasar, serta penggunaan `saveInstanceState` untuk menjaga data saat terjadi perubahan lifecycle. _(6 jam)_  

3. **Intent**: Mendalami berbagai jenis Intent untuk berpindah antar Activity maupun aplikasi, seperti implicit intent dan explicit intent, serta cara mendebug aplikasi jika terjadi error. _(9 jam 20 menit)_  

4. **View dan ViewGroup**: Mempelajari cara mendesain aplikasi dengan berbagai layout, seperti ConstraintLayout, LinearLayout, FrameLayout, RelativeLayout, dan TableLayout. _(6 jam 30 menit)_  

5. **Style dan Theme**: Mempelajari cara mendesain aplikasi lebih cepat dan mudah dengan memanfaatkan Style dan Theme. _(2 jam 40 menit)_  

6. **RecyclerView**: Belajar tentang komponen yang sering digunakan untuk menampilkan list dan cara menambahkan library seperti Glide untuk menampilkan gambar dari internet. _(10 jam 20 menit)_
